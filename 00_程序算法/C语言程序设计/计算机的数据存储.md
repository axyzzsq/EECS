# 计算机中的数据存储

- 正数和负数均以补码的形式存储，输出的时候，补码再做补码运算，可以得到源码

  - principle： 补码的补码是原码

- ```C
  unsigned char item; //范围是0~127(0x0111 1111)
  // 所以在范围内的最高位一定是0,当它是1的时候它就是一个负数
  // 计算机把内存里的一堆010101拿出来的时候，先看最高位是不是1，如果是1则为负数，除了符号位进行取反，低于符号位的全部取反，然后再加一1则得到原码。
  //如果最高位是0，那补码就是原码
  ```
