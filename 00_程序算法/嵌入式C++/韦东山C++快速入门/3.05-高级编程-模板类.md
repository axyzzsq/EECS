# 类模板

## 格式

```c++
/*  
 *  1. 声明  
 */

template<typeclass T>
class AAA 
/* 使用T表示某种类型，比如: */
private:
	T obj;
public:
	void test_func(T& t);
	.....
;

/*  
 *  2. 定义  
 */

template<typeclass T>
void AAA<T>::test_func(T& t)  .... 
```

## 例程1

- 模板类的格式和使用

```c++
#include <iostream>
#include <string.h>
#include <unistd.h>

using namespace std;


template<typename T>
class AAA {
private:
	T t;
public:
	void test_func(const T &t);
	void print(void);
};

template<typename T> void AAA<T>::test_func(const T &t)
{
	this->t = t;
}


template<typename T>
void AAA<T>::print(void)
{
	cout<<t<<endl;
}

int main(int argc, char **argv)
{
	AAA<int> a;

	a.test_func(1);
	a.print();

	AAA<double> b;

	b.test_func(1.23);
	b.print();

	return 0;
}

```
