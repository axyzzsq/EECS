蓝牙/WiFi或者其他具有复杂分层结构的应用开发，关注的核心点是协议栈如何开启，如何关闭。

尤其是关闭。

协议栈的不正确关闭会导致协议栈卡死。如Realtek的BLE协议栈依赖WiFi协议栈底层，当BLE上层应用还在Loop,此时关闭WiFi协议栈，会导致BLE卡死，必须下电重启。

所以要注重协议栈关闭的时序。
