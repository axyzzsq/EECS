# 网络编程概念

1. 数据传输的三要素：源、目的、长度

2. 用`IP`+端口号表示网络通信的源和目的(同一个电脑的连个浏览器的IP地址是一样的，但是端口号不一样，所以要加上目的端口)

3. 服务器端TCP编程框架

   ```C
   fd = socket();  //无三要素
   
   bind(自己的IP，端口号);
   //把fd和自己的IP、端口号绑定起来
   
   liste();  //启动检测数据
   
   accpt();  //接受连接客户端的connect()，建立一条连接
   
   send(); //发送
   
   receive();  //接收

4. 客户端TCP编程框架

   ```c
   fd = socket();
   
   connect();  //建立连接
   
   send();
   
   reveive();
   ```
